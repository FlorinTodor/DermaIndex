# Glosario de Términos del Proyecto DermalIndex

Este documento define los términos clave y conceptos fundamentales utilizados en el proyecto DermalIndex, con especial énfasis en el método de cálculo del propio índice.

---

## DermalIndex (DI)

El DermalIndex es un indicador diseñado para proporcionar una medida del riesgo ambiental para la piel, considerando varios factores clave que pueden afectar condiciones dermatológicas como la dermatitis y/o psoriasis. Su objetivo es ofrecer recomendaciones cualitativas para ayudar a los usuarios a planificar sus actividades diarias y proteger su piel.

### Tipos de DermalIndex: Horario y Diario

El DermalIndex puede ser calculado y presentado de dos maneras principales para satisfacer diferentes necesidades de planificación y análisis:

*   **DermaIndex Horario (`DI_h`):** Es el cálculo fundamental que proporciona un valor de riesgo para cada hora específica. Este formato es ideal para la planificación a corto plazo, permitiendo al usuario identificar momentos óptimos o de alto riesgo dentro de un mismo día.
*   **DermaIndex Diario (`DI_d`):** Representa una agregación o resumen del riesgo para un día completo. Este es útil para una planificación a más largo plazo (ej. planificación semanal) o para obtener una visión general rápida del día.
